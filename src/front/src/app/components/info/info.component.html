For whom it is intended?

This service is for Node JS backend-developers.

 What does the service allow you to do?

With this service you can create public or private (needed subscription) repositories. There you can store a templates for controllers, services, models.

 How it works?

You create a repository. Empty templates for the controller, service and model are automatically created with the repository

 What does a private repository give you?

A private repository allows you to make your templates inaccessible to others.

 Can i have a repository without a subscription?

Yes, but your template will be public.

 Can i share my private repository to others?

Yes, you just need to know a user nickname.

 How create a repository?

Toolbar -> Repositories -> My repositories -> Create
